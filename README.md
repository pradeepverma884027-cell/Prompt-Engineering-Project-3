# Context Anchored Question Answering System

## Overview

This project is a Flask-based AI application that answers user questions using only a provided reference document.

The system uses Prompt Engineering and Context Anchoring to reduce hallucinations.

---

## Features

- Context Anchored Answering
- Closed-book QA
- Gemini API
- Flask Web Application
- Keyword Retrieval
- Response Validation
- Source Citation

---

## Technologies

- Python
- Flask
- Google Gemini
- HTML
- CSS
